## 📢 프로젝트를 통해 배운 점 및 느낀점

<br>
최근영 : 배운 내용을 최대한 활용하여 쥐어 짜낼 수 있는모든 것을 쏟아 부은 것 같다. 실제 부트스트랩 템플릿을 가져와서 적용해 보며 CSS, JS 를 공부해야할 필요성을 느낀 것 같다
    

<br>
이수영 : 저번 실습보다 깃허브를 사용하는 것이 많이 익숙해졌습니다. 팀프로젝트를 통해서 제가 많이 모르는 부분이나 부족한 부분을 팀원들이 채워주셔서 즐겁게 했습니다.


<br>
